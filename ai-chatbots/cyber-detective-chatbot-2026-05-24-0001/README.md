# Cyber Detective Chatbot

A small browser-based AI-style chatbot with a cyber detective theme. It runs fully in the browser and uses a simple intent engine to answer messages like a digital investigator.

## Files

- `index.html` - chatbot page
- `style.css` - interface styling
- `script.js` - chat behavior and themed responses

## Run

Open `index.html` in a browser.

## Try asking

- `scan this clue`
- `who are you?`
- `help me solve a case`
- `give me a cyber tip`
