# ELISA OD Data Analyzer

**Portfolio:** Computational Biotechnology Portfolio 
**Track:** Lab Data Analysis 
**Author:** Meena Vignesh M

## Purpose
This project turns a small biotechnology or healthcare-data task into a reproducible Python workflow. I built it to practice both the science concept and the coding steps clearly.

## Biology / Data Concept
ELISA plate readings use optical density values to compare relative signal across samples.

## How To Run

```bash
python elisa_od_data_analyzer.py sample_elisa.csv
```

## Files
- `elisa_od_data_analyzer.py` - main Python script
- `sample_elisa.csv` - sample input data
- `example_output.txt` - expected output example
