# Lab Report Table Cleaner

**Portfolio:** Computational Biotechnology Portfolio 
**Track:** Lab Data Analysis 
**Author:** Meena Vignesh M

## Purpose
This project turns a small biotechnology or healthcare-data task into a reproducible Python workflow. I built it to practice both the science concept and the coding steps clearly.

## Biology / Data Concept
Lab data becomes easier to review when messy report tables are converted into structured rows and columns.

## How To Run

```bash
python lab_report_table_cleaner.py messy_table.txt
```

## Files
- `lab_report_table_cleaner.py` - main Python script
- `messy_table.txt` - sample input data
- `example_output.txt` - expected output example

## Job Skill Demonstrated
- Python scripting
- CSV/FASTA data handling
- Basic biotechnology interpretation
- Reproducible command-line workflow
- Scientific reporting

## Result
This tool converts raw biological/lab data into a clean summary report.
