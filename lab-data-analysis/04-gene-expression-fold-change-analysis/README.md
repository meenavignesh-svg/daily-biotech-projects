## Purpose
This project compares control and treatment gene expression values and reports fold-change status. I built it to practice a biotechnology data-analysis workflow that is useful for interpreting simple experimental tables.

## Biology / Data Concept
Fold change compares expression between control and treatment conditions. A value above 1 means expression increased in treatment, while a value below 1 means expression decreased.

## How To Run

```bash
python gene_expression_fold_change.py sample_expression.csv
```
