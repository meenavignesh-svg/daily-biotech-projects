## Purpose
This project converts a protein sequence into a clean property summary. I built it to practice connecting amino acid sequence data with useful beginner-level protein interpretation.

## Biology / Data Concept
Protein properties can be estimated from amino acid composition and residue masses. This is not a replacement for advanced protein analysis, but it is a strong starting point for learning sequence-based biotechnology tools.

## How To Run

```bash
python protein_property_calculator.py sample_protein.txt
```

## Files
- `protein_property_calculator.py` - main Python script
- `sample_protein.txt` - sample protein sequence
- `example_output.txt` - expected output example
