## Purpose
This project checks beginner PCR primer quality using simple sequence rules. I built it to practice turning a molecular biology concept into a small reproducible Python report.

## Biology / Data Concept
PCR primers are often screened by length, GC content, and melting temperature before experiments. These checks help decide whether a primer should pass basic review or needs redesign.

## How To Run

```bash
python primer_quality_checker.py sample_primers.csv
```

## Files
- `primer_quality_checker.py` - main Python script
- `sample_primers.csv` - sample primer input data
- `example_output.txt` - expected output example
