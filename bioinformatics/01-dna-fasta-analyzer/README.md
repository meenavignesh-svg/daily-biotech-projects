## Purpose
This project turns a small biotechnology or healthcare-data task into a reproducible Python workflow. I built it to practice both the science concept and the coding steps clearly.

## Biology / Data Concept
DNA sequences can be summarized by base frequency, GC percentage, and RNA transcription output.

## How To Run

```bash
python dna_fasta_analyzer.py sample_sequences.fasta
```

## Files
- `dna_fasta_analyzer.py` - main Python script
- `sample_sequences.fasta` - sample input data
- `example_output.txt` - expected output example
